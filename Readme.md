# Image-to-Speech: Image Captioning and Speech Synthesis

This project uses **BLIP (Bootstrapped Language-Image Pretraining)** to generate captions for images and converts the captions to speech using a Text-to-Speech (TTS) engine. It integrates cutting-edge AI for computer vision and natural language processing to bridge the gap between visual data and audio output.

---

## Features

- **Image Captioning** : Generate natural language captions for images using a fine-tuned BLIP model.
- **Speech Synthesis** : Convert the generated captions into speech using a Text-to-Speech engine.
- **Custom Training** : Train the captioning model on your custom dataset of images and captions.
- **Easy Deployment** : Deploy the model using Flask for inference.
